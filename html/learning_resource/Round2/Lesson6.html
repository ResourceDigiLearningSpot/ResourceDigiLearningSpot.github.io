Bài học: Giải thích và tối ưu thuật toán

Trong bài học này, chúng ta sẽ tìm hiểu về các thuật toán và cách tối ưu hóa chúng để đạt hiệu suất tốt nhất.

Giải thích thuật toán

Thuật toán là một tập hợp các bước hoặc quy tắc để giải quyết một vấn đề cụ thể. Hiểu rõ thuật toán giúp chúng ta áp dụng đúng đắn và hiệu quả.

Dưới đây là một ví dụ về thuật toán tìm kiếm nhị phân trong Python:


def binary_search(arr, x):
    low = 0
    high = len(arr) - 1
    mid = 0

    while low <= high:
        mid = (high + low) // 2

        if arr[mid] < x:
            low = mid + 1
        elif arr[mid] > x:
            high = mid - 1
        else:
            return mid
    return -1

Tối ưu thuật toán

Tối ưu hóa thuật toán liên quan đến việc cải thiện hiệu suất và giảm thời gian chạy của thuật toán. Có nhiều kỹ thuật để tối ưu hóa, bao gồm việc giảm độ phức tạp thời gian và không gian.

Dưới đây là một ví dụ về tối ưu hóa thuật toán sắp xếp:


def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr

Bài tập

  1. Viết một thuật toán tìm kiếm tuyến tính để tìm một phần tử trong một danh sách.
  2. Tối ưu hóa thuật toán sắp xếp chèn để cải thiện hiệu suất.
  3. So sánh độ phức tạp thời gian của thuật toán tìm kiếm nhị phân và tìm kiếm tuyến tính.
