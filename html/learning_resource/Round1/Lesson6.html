Bài học: Ôn tập Object và class trong Python

Trong bài học này, chúng ta sẽ ôn tập về hai khái niệm quan trọng trong lập trình hướng đối tượng: Object và class trong Python.

Class trong Python

Class là một khuôn mẫu để tạo ra các đối tượng. Class định nghĩa các thuộc tính và phương thức mà đối tượng của class đó sẽ có. Một class được khai báo bằng từ khóa class.

Dưới đây là một ví dụ về cách khai báo một class trong Python:


class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Hello, my name is {self.name} and I am {self.age} years old."

Object trong Python

Object là một thể hiện cụ thể của một class. Khi một class được định nghĩa, không có bộ nhớ nào được cấp phát cho nó cho đến khi các đối tượng của class đó được tạo ra. Một đối tượng được tạo ra bằng cách gọi class như một hàm.

Dưới đây là một ví dụ về cách tạo một đối tượng từ class Person và cách sử dụng các phương thức của nó:


person1 = Person("Alice", 30)
print(person1.greet())

Tính kế thừa trong Python

Kế thừa cho phép một class con thừa hưởng các thuộc tính và phương thức từ một class cha. Điều này giúp tái sử dụng mã và tạo ra một hệ thống phân cấp lớp học.

Dưới đây là một ví dụ về tính kế thừa trong Python:


class Employee(Person):
    def __init__(self, name, age, employee_id):
        super().__init__(name, age)
        self.employee_id = employee_id

    def display_id(self):
        return f"My employee ID is {self.employee_id}."

Sử dụng class kế thừa:


employee1 = Employee("Bob", 25, "E1234")
print(employee1.greet())
print(employee1.display_id())

Đa hình trong Python

Đa hình là khả năng của một phương thức có thể thực hiện các hành vi khác nhau dựa trên đối tượng gọi nó. Đa hình cho phép các đối tượng thuộc các class khác nhau có thể được xử lý bằng cách sử dụng cùng một giao diện.

Dưới đây là một ví dụ về đa hình trong Python:


class Dog:
    def speak(self):
        return "Woof!"

class Cat:
    def speak(self):
        return "Meow!"

def animal_sound(animal):
    print(animal.speak())

dog = Dog()
cat = Cat()
animal_sound(dog)
animal_sound(cat)
