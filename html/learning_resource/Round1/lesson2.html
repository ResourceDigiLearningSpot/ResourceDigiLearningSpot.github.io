Bài học 2: Xử lý chuỗi và cấu trúc điều khiển

Trong bài học này, chúng ta sẽ tìm hiểu cách xử lý chuỗi và sử dụng các cấu trúc điều khiển cơ bản trong Python.

Xử lý chuỗi (Strings) trong Python

  • Truy cập phần tử trong chuỗi:
    • my_string = "Hello, world!"
print(my_string[0])  # Output: 'H'
  • Cắt chuỗi (Slicing):
    • print(my_string[7:])  # Output: 'world!'
  • Nối chuỗi:
    • str1 = "Hello"
str2 = "world"
concatenated_string = str1 + ", " + str2 + "!"
print(concatenated_string)  # Output: 'Hello, world!'
  • Đảo ngược chuỗi:
    • reversed_string = my_string[::-1]
print(reversed_string)  # Output: '!dlrow ,olleH'

Cấu trúc điều khiển (Control Structures) trong Python

  • Câu lệnh if:
    • x = 10
if x > 5:
    print("x is greater than 5")
elif x == 5:
    print("x is equal to 5")
else:
    print("x is less than 5")
  • Vòng lặp for với break:
    • for i in range(5):
    if i == 3:
        break
    print(i)
  • Vòng lặp while với continue:
    • count = 0
while count < 5:
    count += 1
    if count == 3:
        continue
    print(count)

Bài tập

  1. Viết một chương trình Python để đảo ngược một chuỗi đã cho.
  2. Viết một chương trình Python để kiểm tra một chuỗi có phải là chuỗi Palindrome hay không.
  3. Viết một chương trình Python để tính tổng các số từ 1 đến 100 bằng cả vòng lặp for và while.
